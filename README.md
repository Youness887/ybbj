## üöÄ Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Python 3.8+ (for running backend bots)
- PayPal account (optional, for payment features)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Youness887/ybbj.git
   cd ybbj
   ```

2. **Set up Python environment**
   ```bash
   pip install -r requirements.txt
   ```

3. **Configure environment variables**
   ```bash
   # Create .env file with your configurations
   PAYPAL_CLIENT_ID=your_paypal_client_id
   PAYPAL_CLIENT_SECRET=your_paypal_client_secret
   EMAIL_USERNAME=your_email@gmail.com
   EMAIL_PASSWORD=your_app_password
   ```

4. **Launch the application**
   - Open `index.html` in your browser for the web interface
   - Run Python bots with `python money_maker_bot.py`
